## Current Status: Alpha

Check out the [Changelog](https://v3.wails.io/changelog) for the latest status.

## How You Can Contribute

* Test the latest alpha release and report any issues
* Contribute to documentation and examples
* Participate in discussions and provide feedback on proposed features
* Submit pull requests for bug fixes or new features

## Feedback and Updates

This roadmap is subject to change based on community feedback and project priorities. We'll update it regularly to reflect our progress and any changes in direction.
