## Features

Wails provides native desktop capabilities:

- [Windows](https://v3.wails.io/features/windows/basics) - Multiple windows, frameless windows, positioning
- [Menus](https://v3.wails.io/features/menus/application) - Application menus, context menus, system tray
- [Dialogs](https://v3.wails.io/features/dialogs/file) - File open/save, message dialogs
- [Events](https://v3.wails.io/features/events/system) - Communication between components
- [Bindings](https://v3.wails.io/features/bindings/methods) - Type-safe Go ↔ JavaScript calls
- [Clipboard](https://v3.wails.io/features/clipboard) - Copy/paste operations
- [Drag & Drop](https://v3.wails.io/features/drag-and-drop/files) - File drag and drop
- [Keyboard](https://v3.wails.io/features/keyboard) - Global shortcuts
